Thư mục input dùng để chứa các file document
Để chạy trương trình bạn hãy thêm các tài liệu dưới dạng txt vào thư mục input
Từ ternimal ghõ python main.py để chạy
Một form dialog sẽ hiện ra, bạn chọn đến thư mục input hoặc bạn cũng có thể chọn một thư mục khác miễn là nó có chưa các file txt làm document
Dữ liệu ghi ra các file txt trong thư mục results có phân cấp cùng với file main.py
